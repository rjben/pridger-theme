# Pridger Theme for Visual Studio Code

A **dark theme for VS Code** designed for developers who want **vibrant syntax highlighting**, **high contrast readability**, and **multi-language support**. Perfect for coding in **TypeScript, JavaScript, Python, Rust, Java, C++, PHP, and more**.

## ✨ Key Features
- 🎨 **Modern dark palette** with carefully chosen colors for operators, constants, keywords, and variables
- 🐍 **Python support**: highlights `self`, parameters, and function calls
- 🦀 **Rust support**: lifetimes, language keywords, and std functions styled distinctly
- ☕ **Java support**: imports, modifiers, and type annotations highlighted for clarity
- 📘 **TypeScript & JavaScript support**: console functions, DOM variables, JSON properties, and math constants
- 💻 **C/C++ support**: operators, functions, and block punctuation styled for readability
- 🌐 **Web languages**: CSS, SCSS, HTML, and Markdown with custom colors

## 🚀 Installation
1. Open **Visual Studio Code**
2. Go to **Extensions** (`Ctrl+Shift+X`)
3. Search for **Pridger Theme**
4. Click **Install** and activate the theme
